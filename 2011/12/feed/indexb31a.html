today we have released the final version of Z-Push 1.5.6. It is identical to 1.5.6RC-933 (except the version of course).

Notes

This is a maintenance release for Z-Push 1.5

It is recommended for Zarafa users who use Outlook Today view to track today’s tasks
because now tasks created on a mobile device appear there as well.

We have also added a configurable ping interval parameter.
Setting it to a higher value decreases the server load in certain cases.
It is the highest time span in seconds before a change is sent to mobile.
Be aware that setting it to a value too high might result in a time-out.

Changes since 1.5.5

General
– added: configurable $timeout in HandlePing (Mantis #528)

Zarafa/ICS
– fixed: a task created on a mobile doesn’t appear in Outlook Today view. The due date must be at 12:00AM straight (Mantis #525)

IMAP backend
– fixed: Sending an email the “from” Field in the maillog is <apache> and not real email address (Mantis #497)
