today we have released the Z-Push version 1.5.8 final. It is identical to the version 1.5.8RC-1174.

Notes
This is a maintenance release for Z-Push 1.5.

It is recommended to upgrade if touchdown is used as the android client and the provisioning is disabled.
The upgrade is also advisable if you have appointments with many attendees.
This version also fixes the issue that new task or calendar folders created on iOS 5 are displayed as email folders on the server.

Changes since 1.5.7

General
– fixed: do not sent Provisioning command in OPTIONS if provisioning is disabled (Mantis #603)

Zarafa/ICS
– changed: only process max. 256 attendees of an appointment (Mantis #567)
– fixed: new tasks or calendar folders created with iOS 5 are of type email on the server (Mantis #607)
– fixed: PR_SEARCH_KEY is set to a wrong value (Mantis #570)
