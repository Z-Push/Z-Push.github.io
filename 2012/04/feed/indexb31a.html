today we have released Z-Push 1.5.9 final version. It is identical to Z-Push 1.5.9RC-1202.

Notes
This is a maintenance release for Z-Push 1.5.

It is recommended to update if Z-Push with Zarafa/ICS backend is used on systems with PHP version higher than or equal to 5.4
This version also fixes the issue that it is possible to delete Outbox and some other system folders on iOS 5 devices.

Changes since 1.5.8

General
– fixed: PHP 5.4 has own implementation of hex2bin() (Mantis #633)
– fixed: SYNC_FILTERTIME_MAX is not always honoured (Mantis #632)

Zarafa/ICS
– fixed: iOS 5 allows deletion of Outbox folder (Mantis #621)

DiffBackend
– fixed: typo which prevented conflict detection in DiffBackend/DiffState (Mantis #637)
