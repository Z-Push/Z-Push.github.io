News | Z-Push https://z-push.org Open source push technology Sat, 01 Feb 2014 14:43:20 +0000 en-US hourly 1 https://wordpress.org/?v=5.6.10 Z-Push 1.4.3 final and 1.5 beta2 released https://z-push.org/z-push-1-4-3-final-and-1-5-beta2-released/ https://z-push.org/z-push-1-4-3-final-and-1-5-beta2-released/#respond Fri, 19 Nov 2010 12:00:24 +0000 http://62.245.227.250/?p=75 On 19/11/2010 we have released the final version of Z-Push 1.4.3, a maintenance release for Z-Push 1.4.x. This release is the same as 1.4.3RC. If you already use this version, no upgrade is necessary. This is the latest stable version.

This version is especially recommended for German iPhone iOS4 users. It fixes an issue where outgoing emails with recipients containing umlauts are not sent.
More informations about this issue at:https://developer.berlios.de/mantis/view.php?id=196

On 19/11/2010 we have also released the beta2 of Z-Push 1.5 version. It has configurable state directory, improved results from GAL search and also some fixes for the ICS and IMAP backends.

While upgrading, you MUST adjust the “state-directory” setting in the config file, or your mobile will go out of sync!
If you want to make use of the possibility to store the states in a folder outside of Z-Push, make sure you copy or move the old states into the new location.

The version 1.5.2 beta2 is NOT ready for production environment!
