Hi all,<\/p>\n

today we have released the Z-Push 2.2.2 final version (build 1981).<\/p>\n

Important notes<\/strong>
\nThis version contains several bug fixes and enhancements.
\nWe had to disable checking of notifications validity in Ping. This will generate additional requests, because ping will reply on all received notification regardless the sync interval.
\nIt is now possible to use Outlook 2013 via ActiveSync with Z-Push. Please be aware that this functionality is not yet ready for production use, but is meant as a preview. We are aware of several missing features in Outlook via ActiveSync, like no availability of free-busy data, missing reply & forward flags of emails and the missing Global Address Book (GAB). We have several ideas and plans how to fix these issues.
\nAndroid devices with Cyanogenmod 11 should be able to sync now.
\nThe issue with faulty appointment status on iOS 8 was fixed.
\nEndless sync on Blackberry and some few LG devices was fixed.
\nIncreased the default SYNC_CONTACTS_MAXPICTURESIZE value in config.php.
\nThe issue with the duplicate “RSS Feeds” folder in Outlook 2013 has been fixed.
\nSetting folder display name to “unknown” if it is not available in the store (rare issue).
\nlistfolders.php now accepts ssl certificates for authentication.
\nSome enhancements for z-push-admin.<\/p>\n

Changes since Z-Push 2.2.1<\/strong>
\nGeneral
\n[ZP-451] – Mobile sending a folder type receives wrong ping information.
\n[ZP-544] – Not compatible with Cyanogenmod 11.
\n[ZP-558] – Enhance error message for folder resync.
\n[ZP-563] – z-push-admin ignores -u when -d is given.
\n[ZP-572] – Increase default SYNC_CONTACTS_MAXPICTURESIZE.
\n[ZP-604] – Allow OL connections via ActiveSync.
\n[ZP-606] – Warnings in appointment sync.
\n[ZP-608] – Write binary logs.
\n[ZP-611] – Set folder display name to “unknown” if the name is not available.
\n[ZP-623] – Resync folder if ICS exports <Remove> requests for all items of a folder.
\n[ZP-631] – Disable checking of notifications validity in Ping.<\/p>\n

Zarafa
\n[ZP-554] – extend listfolders.php for ssl certificates.
\n[ZP-586] – Appointments accepted in Outlook are not accepted on iOS 8.
\n[ZP-588] – Log sourcekey, parentsourcekey and entryid before retrieving message from the MAPIProvider.
\n[ZP-589] – Add application version\/type and client version\/type to mapi_logon call.
\n[ZP-607] – “RSS Feeds” seen twice in OL 2013 via AS.<\/p>\n

The version is available at: http:\/\/download.z-push.org\/final\/2.2\/<\/a>.<\/p>\n

Your feedback is highly appreciated.<\/p>\n

Kind regards,
\nManfred
