News | Z-Push https://z-push.org Open source push technology Sat, 01 Feb 2014 14:41:05 +0000 en-US hourly 1 https://wordpress.org/?v=5.6.10 Z-Push 1.3 released https://z-push.org/z-push-1-3-released/ https://z-push.org/z-push-1-3-released/#respond Thu, 27 May 2010 12:00:24 +0000 http://62.245.227.250/?p=69 On 22/04/2010 we have released the final version of Z-Push version 1.3.

This version introduces the remote wipe functionality.
You can disable remote wipe by setting the PROVISIONING parameter in config.php.

Devices can be adminstrated using a Zarafa Webaccess Plugin available the Zarafa homepage.

A command line tool will be introduced in Z-Push 1.4 which is already available as beta.
