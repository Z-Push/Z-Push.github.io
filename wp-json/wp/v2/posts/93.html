Hi,<\/p>\n

today we have released Z-Push 1.5.9 final version. It is identical to Z-Push 1.5.9RC-1202.<\/p>\n

Notes
\nThis is a maintenance release for Z-Push 1.5.<\/p>\n

It is recommended to update if Z-Push with Zarafa\/ICS backend is used on systems with PHP version higher than or equal to 5.4
\nThis version also fixes the issue that it is possible to delete Outbox and some other system folders on iOS 5 devices.<\/p>\n

Changes since 1.5.8<\/strong><\/p>\n

General
\n– fixed: PHP 5.4 has own implementation of hex2bin() (Mantis #633)
\n– fixed: SYNC_FILTERTIME_MAX is not always honoured (Mantis #632)<\/p>\n

Zarafa\/ICS
\n– fixed: iOS 5 allows deletion of Outbox folder (Mantis #621)<\/p>\n

DiffBackend
\n– fixed: typo which prevented conflict detection in DiffBackend\/DiffState (Mantis #637)<\/p>\n

The version is available at:\u00a0http:\/\/z-push.sourceforge.net\/download<\/a><\/p>\n

Your feedback is highly appreciated.<\/p>\n

Kind regards,
\nManfred
