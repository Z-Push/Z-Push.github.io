Z-Push 2.3.9 final release

by Admin | Feb 7, 2018 | News | 2 comments

Hi all,

we have just released Z-Push 2.3.9 final (tag 2.3.9, commit e598af87df75d47f6f0e74b121fc60474a73de84).
This version is released without a previous beta due to the update of 2.3.8 which might not have been installed on all distributions.

It’s a maintenance release which contains only the update fix of 2.3.8 (ZP-1339) and fixes issues with private items in shared folders.

If you are running KC 8.4.90 or later you have to update Z-Push to this version.

The full changelog is available at http://download.z-push.org/final/2.3/z-push-2.3.9.txt and at the forum.

Code changes since Z-Push 2.3.8: https://stash.z-hub.io/projects/ZP/repos/z-push/compare/diff?targetBranch=refs%2Ftags%2F2.3.8&sourceBranch=refs%2Ftags%2F2.3.9

Install Z-Push 2.3.9 from the final repositories (recommended): https://wiki.z-hub.io/display/ZP/Installation

The tarball is available here (not recommended): http://download.z-push.org/final/2.3/z-push-2.3.9.tar.gz

Please share your experiences and give feedback!

Greetings,
Sebastian
Z-Push dev team
