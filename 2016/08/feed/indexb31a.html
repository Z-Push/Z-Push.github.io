today we have released the Z-Push 2.2.12 final version (tag 2.2.12, commit 61739c5eb9e84e742b8b804be5af3c4dbe055422).
Once again we thank all contributors for their work on this release.

Important notes
This version contains some fixes and improvements.

Meetings on iOS could not be accepted under certain circumstances and exceptions of recurring all day events were no all day events anymore (from 00:00 to 23:59 with variations of time difference to GMT before the fix).
Also if you deleted an occurrence of an all day event it was not correctly deleted on the server.

Some other improvements.

The full change log is available here.

If you are interested in a detailed overview of the code changes, see https://stash.z-hub.io/projects/ZP/repos/z-push/compare/diff?targetBranch=refs%2Ftags%2F2.2.11&sourceBranch=refs%2Ftags%2F2.2.12

The version is available at: http://download.z-push.org/final/2.2/.

Your feedback is highly appreciated.

Kind regards,
Manfred
Z-Push dev team
