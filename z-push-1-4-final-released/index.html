Z-Push 1.4 final released

by Admin | Jun 18, 2010 | News | 0 comments

On 14/06/2010 we have released the final version of Z-Push 1.4.

This version contains the remote wipe command line script. You can see the status of devices and request the remote wipe.

It also contains several fixes and changes for Mail4Exchange 3. If your M4E broke with previous version, you need to reconfigure your profile on the mobile.

Several issues with recurring Meeting Requests on the mobiles were fixed.

Greets, Manfred
Z-Push dev team
