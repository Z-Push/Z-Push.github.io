Hi,<\/p>\n

today we have released the final version of Z-Push 1.5.6. It is identical to 1.5.6RC-933 (except the version of course).<\/p>\n

Notes<\/strong><\/p>\n

This is a maintenance release for Z-Push 1.5<\/p>\n

It is recommended for Zarafa users who use Outlook Today view to track today’s tasks
\nbecause now tasks created on a mobile device appear there as well.<\/p>\n

We have also added a configurable ping interval parameter.
\nSetting it to a higher value decreases the server load in certain cases.
\nIt is the highest time span in seconds before a change is sent to mobile.
\nBe aware that setting it to a value too high might result in a time-out.<\/p>\n

Changes since 1.5.5<\/strong><\/p>\n

General
\n– added: configurable $timeout in HandlePing (Mantis #528)<\/p>\n

Zarafa\/ICS
\n– fixed: a task created on a mobile doesn’t appear in Outlook Today view. The due date must be at 12:00AM straight (Mantis #525)<\/p>\n

IMAP backend
\n– fixed: Sending an email the “from” Field in the maillog is <apache> and not real email address (Mantis #497)<\/p>\n

The version is available at:\u00a0http:\/\/z-push.sourceforge.net\/download<\/a><\/p>\n

Your feedback is highly appreciated.<\/p>\n

Kind regards,
\nManfred
