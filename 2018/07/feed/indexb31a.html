News | Z-Push https://z-push.org Open source push technology Wed, 25 Jul 2018 20:05:35 +0000 en-US hourly 1 https://wordpress.org/?v=5.6.10 Z-Push 2.4.4 final release https://z-push.org/z-push-2-4-4-final-release/ https://z-push.org/z-push-2-4-4-final-release/#respond Wed, 25 Jul 2018 20:05:35 +0000 http://z-push.org/?p=613 Hello,

we have just released Z-Push 2.4.4 final (tag 2.4.4 commit fa0c7a7f1b87432a178066cf8869e0032abc03d4).

This is a maintenance release with some bugfixes and improvements.

Michele has contributed a solution for saving states on environments with high traffic (e.g. file state on NFS storage). Check https://wiki.z-hub.io/display/ZP/Hidden+configuration+parameters for the usage instructions.

Michele also added a better handling of sending emails when the store is full.

Some Android devices weren’t able to download attachments. This issue has been fixed.

We also resolved the issue of an occurrence of a recurring event not being deleted in certain cases.

The tracking tab of a meeting with multiple attendees would disappear in webapp when an ActiveSync device is connected. Thanks to Mathias for reporting and providing steps to reproduce this.

Install Z-Push 2.4.4 from the final repositories (recommended) Z-Push Wiki.

The complete changelog is available here.

Code changes since Z-Push 2.4.3: https://stash.z-hub.io/projects/ZP/repos/z-push/compare/diff?targetBranch=refs%2Ftags%2F2.4.3&sourceBranch=refs%2Ftags%2F2.4.4

Please share your experiences (positive and negative) and give feedback!

Greetings,
Manfred
