Z-Push 2.5.2 final released

by Admin | Mar 26, 2020 | News | 6 comments

Hello,

we have just released Z-Push 2.5.2 final (tag 2.5.2 commit 0b02fe87b129a46f16286c2a566d0ee41da5561a).

This maintainence release includes several bug fixes.

Outlook over ActiveSync would not sync if “mobile” feature was disabled. Now only “outlook” feature is required for it.

Group Everyone was still being added as member of group Everyone on multi tenant systems. This has been fixed.

In certain cases accepting meeting requests on an iOS devices did not work.

Forwarding an email with Nine client did not include the original message.

Install Z-Push 2.5.2 final from the final repositories (recommended) Z-Push Wiki.

Tickets in Z-Push 2.5.2

Bug
[ZP-1519] – Outlook feature doesn’t work when “mobile” feature is disabled
[ZP-1524] – Accepting meeting request on an ios device does not work
[ZP-1526] – [gabsync] Everyone is member of Everyone on multi tenant systems
[ZP-1531] – Forwarding email doesn’t include the forwarded message

Improvement
[ZP-1515] – Document possible IPC_PROVIDER values

Code changes since Z-Push 2.5.1: https://stash.z-hub.io/projects/ZP/repos/z-push/compare/diff?targetBranch=refs%2Ftags%2F2.5.1&sourceBranch=refs%2Ftags%2F2.5.2

Please share your experiences (positive and negative) and give feedback!

Greetings,
Manfred
