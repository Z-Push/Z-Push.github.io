On 26\/05\/2011 we have released the final version of Z-Push 1.5.3.<\/p>\n

This is maintenance release for Z-Push 1.5.<\/p>\n

This release is recommended for all Zarafa users which extensively use meeting requests on their mobiles.
\nIt is also recommended for the IMAP backend users as it fixes some issues regarding forwarding emails.<\/p>\n

NOTE: we are changing the naming convention starting with this release. From now on it will have the following structure:
\nz-push-[version]-{buildnr}.tar.gz\u00a0and\u00a0z-push-[version]-{buildnr}\u00a0for the start folder inside the archive.<\/p>\n

For example: z-push-1.5.3RC-627.tar.gz with the start folder z-push-1.5.3RC-627 in it.<\/p>\n

The final version won’t have any tag. For example: z-push-1.5.3-685.tar.gz with the start folder z-push-1.5.3-685 in it.<\/p>\n

The full changelog is available\u00a0here<\/a>.<\/p>\n

Download at:\u00a0http:\/\/z-push.sourceforge.net\/download<\/a><\/p>\n

Your feedback is highly appreciated.<\/p>\n

Greets, Manfred
