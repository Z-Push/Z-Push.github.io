Z-Push 2.3.4 final released

by Admin | Dec 21, 2016 | News | 1 comment

Hi all,

we have just released Z-Push 2.3.4 final version (tag 2.3.4, commit cb60b14dacebef1aefba404a1b05810e3b1d27b2).

This version is PHP 7 compatible and introduces many other important changes.
The PHP 7 compatibility is focussed on the main Z-Push code, there can be issues with specific backends or backend functionality.
We would like to thank all our community contributors and everyone who tests Z-Push and provides us feedback. Please continue doing that.

Important notes
The PHP 7 compatibility is focussed on the main Z-Push code, there can be issues with specific backends or backend functionality. For Kopano you require Kopano Core 8.2 which is not yet released as stable.

Z-Push 2.3.2 already contains fixes related to Meeting Requests on iOS. While this fix works, it’s not backward compatible to ZCP 7.1.x and 7.2.x before 7.2.3 versions.
This release fixes it also for these older versions.

We did introduce two new configuration options and did some changes in the headers. These will require you to update your configuration files.

This version is required for KOE 1.2 which will released soon. It contains an improved API to open shared folders, also in a hierarchical expected way.
It’s now also possible to stop synchronizing hidden GAB objects to the KOE GAB. See here how to do this: https://wiki.z-hub.io/display/ZP/Kopano+OL+Extension#KopanoOLExtension-HiddenGABentries

The full changelog is available here.

Code changes since Z-Push 2.3.3: https://stash.z-hub.io/projects/ZP/repos/z-push/compare/diff?targetBranch=refs%2Ftags%2F2.3.3&sourceBranch=refs%2Ftags%2F2.3.4

Install Z-Push 2.3.4 from the final repositories (recommended): https://wiki.z-hub.io/display/ZP/Installation

The tarball is available here (not recommended): http://download.z-push.org/final/2.3/z-push-2.3.4.tar.gz

Please share your experiences and give feedback!

Greetings,
Manfred
Z-Push dev team
