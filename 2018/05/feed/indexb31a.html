we have just released Z-Push 2.4.2 final** (tag 2.4.2 commit e2a70a79747c8004701f7c915b18587b79f7c59d).

This is a maintenance release with some bugfixes and improvements.

We have improved Z-Push packages. For debian-based systems Felix changed the dependency handling.
RHEL-based systems now should have the correct permissions and ownership for state and log folders. Please check carefully if everything works as expected.

In certain cases Z-Push triggers folder re-sync if the client requests deletion a lot of items. Sometimes this a desired behaviour and the folder re-sync is not necessary. Michele added new feature to take a threshold into account. Check [Z-Push Wiki](https://wiki.z-hub.io/display/ZP/Hidden+configuration+parameters) for more information.

For the Kopano backend we fixed saving picture for a contact created on mobile and unsetting properties for tasks.

Niels added authentication with GSSAPI support for IMAP servers. Therefore changes in the config of the IMAP backend might be necessary.

Bart fixed several bugs for imap and caldav backends.

Thank you all for the contributions!

Install Z-Push 2.4.2 from the final repositories (recommended) https://wiki.z-hub.io/display/ZP/Installation.

The complete changelog is available here.

Code changes since Z-Push 2.4.1: https://stash.z-hub.io/projects/ZP/repos/z-push/compare/diff?targetBranch=refs%2Ftags%2F2.4.1&sourceBranch=refs%2Ftags%2F2.4.2

Please share your experiences (positive and negative) and give feedback!

Greetings,
Manfred

]]> https://z-push.org/z-push-2-4-2-final-release/feed/ 0 Z-Push 2.3 end of life https://z-push.org/z-push-2-3-end-of-life/ https://z-push.org/z-push-2-3-end-of-life/#respond Mon, 14 May 2018 15:22:14 +0000 http://z-push.org/?p=602 Hi all,

if you are already using some Z-Push 2.4.X version you can skip this announcement. If you are running Z-Push 2.3.X, please plan an upgrade to the most recent Z-Push 2.4.

We will deprecate Z-Push 2.3 on the 30th of June 2018. Its last stable version 2.3.9 was released in February and already contained only important fixes backported from Z-Push 2.4. The active development on 2.3 ceased about 6 months ago. If there are any issues with 2.3 versions, we will fix them as part of some 2.4 release. There will be no further 2.3.X releases. We will of course provide upgrade support after the deprecation day.

For now we don’t have any plans to remove the downloads of the old versions. The old-final repository will continue as well, until there will be Z-Push 2.5 release. At this point we will move Z-Push 2.4 to old-final.

The upgrade from 2.3 to 2.4 is pretty painless and there are a lot of new features and improvements. Try it out if you haven’t yet!

Manfred
