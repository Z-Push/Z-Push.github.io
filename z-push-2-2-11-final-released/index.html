Z-Push 2.2.11 final released

by Admin | Jul 7, 2016 | News | 0 comments

Hi all,

today we have released the Z-Push 2.2.11 final version (tag 2.2.11, commit 2f61f031c416b7b78c85f02ff86454e394280888). Once again we thank all contributors for their work on this release.

Important notes
This version contains some fixes and improvements.

Ping now should catch all notifications when there are shared folders configured.
Calling z-push-top now works from any directory.
There was a bug in handling timezone information which caused some appointments to have wrong time.
When trying to move a message to trash that is not on the server anymore, we now tell the mobile that worked to prevent incompatibilities.
The GetHierarchy() for old ActiveSync versions was fixed. The printwbxml debug script can now be executed from any folder.
Some other improvements.

The full change log is available here.

The version is available at: http://download.z-push.org/final/2.2/.

Your feedback is highly appreciated.

Kind regards,
Manfred
Z-Push dev team
