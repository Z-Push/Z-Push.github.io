today we have released Z-Push 1.5.10 final version. It is identical to Z-Push-1.5.10RC-1246.

Notes

This is a maintenance release for Z-Push 1.5.

It is recommended to update if Z-Push with Zarafa/ICS backend and Android phones. Inline images were sometimes not available for download.
It also fixes an issue with double encoded names with high caracters in email addresses which caused iOS to display these names with a broken encoding.

Changes since 1.5.9

Zarafa/ICS
[ZP-4] – Double quoted email addresses show invalid high character encoding on iOS
[ZP-61] – Inline.txt instead of inline image on Android

The version is available at: http://z-push.sourceforge.net/download

Your feedback is highly appreciated.

Kind regards,
Manfred
