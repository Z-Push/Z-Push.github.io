Hi all,<\/p>\n

today we have released the Z-Push 2.2.11 final version (tag 2.2.11, commit 2f61f031c416b7b78c85f02ff86454e394280888). Once again we thank all contributors for their work on this release.<\/p>\n

Important notes<\/strong>
\nThis version contains some fixes and improvements.<\/p>\n

Ping now should catch all notifications when there are shared folders configured.
\nCalling z-push-top now works from any directory.
\nThere was a bug in handling timezone information which caused some appointments to have wrong time.
\nWhen trying to move a message to trash that is not on the server anymore, we now tell the mobile that worked to prevent incompatibilities.
\nThe GetHierarchy() for old ActiveSync versions was fixed. The printwbxml debug script can now be executed from any folder.
\nSome other improvements.<\/p>\n

The full change log is available here<\/a>.<\/p>\n

The version is available at: http:\/\/download.z-push.org\/final\/2.2\/<\/a>.<\/p>\n

Your feedback is highly appreciated.<\/p>\n

Kind regards,
\nManfred
