Hi all,<\/p>\n

today we have released the Z-Push 2.1.2 final version (build 1873). It is the same as the 2.1.2RC (except the build number of course).<\/p>\n

Important notes<\/strong>
\nThis release fixes several processing issues.
\nAndroid 4.4 devices are now able to open emails and attachments.
\nA rare issue that meeting attendees are twice on iOS devices also has been fixed.
\nLG Android devices should now be able to sync using Z-Push.<\/p>\n

A new webservice interface to list all existing users and devices on a Z-Push host was introduced.
\nIt can be used by administration interfaces to provide mobile management functionality.
\nMore information about this interface, can be found at: https:\/\/jira.zarafa.com\/browse\/ZP-462<\/a>
\nThis feature is disabled by default.<\/p>\n

Changes since Z-Push 2.1.1<\/strong>:<\/p>\n

New features
\n[ZP-462<\/a>] – Enhance Z-Push webservice api to give list of users<\/p>\n

General
\n[ZP-467<\/a>] – Process ItemOperations:Schema tag
\n[ZP-477<\/a>] – Interpret WindowsSize of 0 as 512
\n[ZP-481<\/a>] – Empty Supported tag breaks synchronization
\n[ZP-493<\/a>] – Z-Push state version file is broken
\n[ZP-494<\/a>] – LG-D802 stops syncing after Settings command<\/p>\n

Zarafa
\n[ZP-489<\/a>] – Attendee of MR is duplicated<\/p>\n

The version is available at: http:\/\/download.z-push.org\/final\/2.1\/<\/a>.<\/p>\n

Your feedback is highly appreciated.<\/p>\n

Kind regards,
\nManfred
