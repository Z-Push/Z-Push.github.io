FAQ

Using any kind of software leaves some questions unanswered. Here we will try to resolve the most common issues…

Further information about devices and compatibility status can be found in the Mobile Compatibility List.

  1. It isn’t working!
  2. Does it it work with my device?
  3. How do I enable the WBXML log?

1. It isn’t working!

Please check at first, if your webserver is working correctly. Open the following URL in your web browser:

https://[your-server-name]/Microsoft-Server-ActiveSync

An authentication dialog should be shown. You should be able to log in with your normal credentials.

If the dialog isn’t shown, something is wrong with your installation. Re-check the instructions in the INSTALL file or ask in theÂ forum.

2. Does it work with my device?

In general, Z-Push works with all ActiveSync compatible mobile devices, including Windows Phones, iPhones, Android, Sony Ericsson and Nokia mobile devices. A working internet connection to your Z-Push enabled server is necessary.

Detailed information about devices and compatibility status can be found in the Mobile Compatibility List.

You are welcome to give feedback and suggestions about the device compatibility so we can improve the compatibility list.

3. How do I enable the WBXML log?

In Z-Push 2.X versions set the value of LOGLEVEL parameter in the config to LOGLEVEL_WBXML. The log file stays the same.
