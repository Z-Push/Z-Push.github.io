Hi all,<\/p>\n

today we have released the Z-Push 2.2.14 final version (tag 2.2.14, commit e2c399567fdbdaa8775457667b371559eda4d7ff).
\nOnce again we thank all contributors for their work on this release.<\/p>\n

Release notes<\/strong>
\nThis is the last release of the Z-Push 2.2.x series. From 2017 on we will give only upgrade support for these versions.
\nDetails can be found on: https:\/\/wiki.z-hub.io\/display\/ZP\/Version+lifecycle<\/a><\/p>\n

Meeting requests on iOS<\/strong>
\nZ-Push 2.2.13 already contains fixes related to Meeting Requests on iOS. While this fix works, it’s not backward compatible to ZCP 7.1.x and 7.2.x before 7.2.3 versions.
\nThis release fixes it also for these older versions.<\/p>\n

Outlook over ActiveSync<\/strong>
\nIf you use (or want to use) Outlook over ActiveSync with Z-Push this will not work with this version anymore.
\nWe decided to block Outlook access, because of many issues that are solved in Z-Push 2.3.
\nIf you want to use Outlook, please upgrade to the latest Z-Push 2.3 version.
\nUpgrade information can be found here: https:\/\/wiki.z-hub.io\/display\/ZP\/Upgrade+to+Z-Push+2.3<\/a><\/p>\n

In this final release we also added the WebserviceInfo->About() call which allows WebApp (or other system using Z-Push Webservices) to display Z-Push version.<\/p>\n

The full change log is available here<\/a>.<\/p>\n

If you are interested in a detailed overview of the code changes, see https:\/\/stash.z-hub.io\/projects\/ZP\/repos\/z-push\/compare\/diff?targetBranch=refs%2Ftags%2F2.2.13&sourceBranch=refs%2Ftags%2F2.2.14<\/a><\/p>\n

The version is available at: http:\/\/download.z-push.org\/final\/2.2\/<\/a>.<\/p>\n

Your feedback is highly appreciated.<\/p>\n

Kind regards,
\nManfred
