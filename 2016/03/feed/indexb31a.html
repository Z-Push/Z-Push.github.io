today we have released the Z-Push 2.2.9 final version (tag 2.2.9, commit a2eef6ce5f9e7fcf720d223f557123b9934dd4f6). Once again we thank all contributors for their work on this release.

Important notes
This version contains some fixes and improvements.

We have changed the default MAPI server location for the Zarafa backend. This was due because of configuration file changes in ZCP 7.2.2. Please check the MAPI_SERVER value in config.php in backend/zarafa folder and modify it if necessary to fit your needs.

It should be possible now to have multiple accounts on a device with Zimbra backend. Cleanup of these backends states did not work correctly in 2.2.9beta1.
The issues with appointments’ and tasks’ privacy on the Zarafa backend and some android clients should work as expected now.
The modification time of a note is now processed correctly and the notes’ order on iphone devices is chronological.

Several other fixes to increase user experience and performance.

The full change log is available here.

The version is available at: http://download.z-push.org/final/2.2/.

Your feedback is highly appreciated.

Kind regards,
Manfred
Z-Push dev team
