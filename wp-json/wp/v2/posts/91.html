Hi,<\/p>\n

today we have released the Z-Push version 1.5.8 final. It is identical to the version 1.5.8RC-1174.<\/p>\n

Notes
\nThis is a maintenance release for Z-Push 1.5.<\/p>\n

It is recommended to upgrade if touchdown is used as the android client and the provisioning is disabled.
\nThe upgrade is also advisable if you have appointments with many attendees.
\nThis version also fixes the issue that new task or calendar folders created on iOS 5 are displayed as email folders on the server.<\/p>\n

Changes since 1.5.7<\/strong><\/div>\n

\nGeneral
\n– fixed: do not sent Provisioning command in OPTIONS if provisioning is disabled (Mantis #603)<\/p>\n

Zarafa\/ICS
\n– changed: only process max. 256 attendees of an appointment (Mantis #567)
\n– fixed: new tasks or calendar folders created with iOS 5 are of type email on the server (Mantis #607)
\n– fixed: PR_SEARCH_KEY is set to a wrong value (Mantis #570)<\/p>\n

The version is available at:\u00a0http:\/\/z-push.sourceforge.net\/download<\/a><\/p>\n

Your feedback is highly appreciated.<\/p>\n

Kind regards,
\nManfred
