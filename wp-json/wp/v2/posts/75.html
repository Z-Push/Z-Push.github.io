On 19\/11\/2010 we have released the final version of Z-Push 1.4.3, a maintenance release for Z-Push 1.4.x. This release is the same as 1.4.3RC. If you already use this version, no upgrade is necessary. This is the latest stable version.<\/p>\n

This version is especially recommended for German iPhone iOS4 users. It fixes an issue where outgoing emails with recipients containing umlauts are not sent.
\nMore informations about this issue at:https:\/\/developer.berlios.de\/mantis\/view.php?id=196<\/a><\/p>\n

On 19\/11\/2010 we have also released the beta2 of Z-Push 1.5 version. It has configurable state directory, improved results from GAL search and also some fixes for the ICS and IMAP backends.<\/p>\n

While upgrading, you MUST adjust the “state-directory” setting in the config file, or your mobile will go out of sync!
\nIf you want to make use of the possibility to store the states in a folder outside of Z-Push, make sure you copy or move the old states into the new location.<\/p>\n

The version 1.5.2 beta2 is NOT ready for production environment!<\/strong><\/p>\n

Download at:\u00a0http:\/\/z-push.sourceforge.net\/download<\/a><\/p>\n

Greets, Manfred
