Hi all,<\/p>\n

we have just released Z-Push 2.3.7 final (tag 2.3.7, commit d788cf9113e21747791bcfd5f9c7261958e20ead).<\/p>\n

This is a maintenance release with several improvements for KOE, but also of general nature.
\nBesides one minor ticket (ZP-1243) it is the same as 2.3.7 beta1.<\/p>\n

For KOE 1.4 Z-Push now supports the request of delivery receipts and improved handling of private appointments.
\nA new KOE capability flag was added to the main Z-Push configuration file. Please update your config accordingly.<\/p>\n

Synchronization of private appointments<\/strong>
\nPrivate appointments are now not being ignored in shared folders anymore. During the synchronization process their data is stripped, which means that private appointments show up in a shared folder, but have no sensitive data. It\u2019s not possible to edit (save) these appointments via shared access.
\nOlder KOE versions will allow editing these in Outlook, but the data will not be saved on the server.
\nCalendars that are already shared need to be re-synchronized in order to see existing private appointments (close & re-open shared calendar).<\/p>\n

Prioritization of the KOE GAB<\/strong>
\nAll KOE versions will benefit from the prioritization of the KOE GAB. Until now it could happen that the GAB was only available at the end of the synchronization. This will now happen early in the synchronization process.<\/p>\n

KOE 1.4<\/strong>
\nPlease be aware that for many of the new features you need KOE 1.4, the release candidate (RC) can be found here: https:\/\/kopano.com\/kopano-ol-extension-1-4-rc-now-available\/<\/a><\/p>\n

There have also been some contributions from Chris (IPv6 logging) and Vincent (loosen SyncObject.equals) in this version. Thanks guys!<\/p>\n

This version has also a repository for upcoming Debian 9 available here: http:\/\/repo.z-hub.io\/z-push:\/final\/Debian_9.0\/<\/a><\/p>\n

The full changelog is available at http:\/\/download.z-push.org\/final\/2.3\/z-push-2.3.7.txt<\/a> and at the forum<\/a>.<\/p>\n

Code changes since Z-Push 2.3.6: https:\/\/stash.z-hub.io\/projects\/ZP\/repos\/z-push\/compare\/diff?targetBranch=refs%2Ftags%2F2.3.6&sourceBranch=refs%2Ftags%2F2.3.7<\/a><\/p>\n

Install Z-Push 2.3.7 from the final repositories (recommended): https:\/\/wiki.z-hub.io\/display\/ZP\/Installation<\/a><\/p>\n
