today we have released the Z-Push 2.0.8 final version (build 1731).

Notes
This version contains several fixes. Changing note of an occurrence of a
recurring appointment now is synced to the mobile. Also an issue that the
tracking tab in the WebApp was missing for meetings created on iOS devices
has been resolved.
Emails flagged on an Iphone are now correctly displayed in the task view.
Reminder is no longer active for completed tasks.
IOS devices now are able to download signed emails with large attachments.
Better loging handling.

Changes

General
[ZP-419] – Typo in syncappointment.php
[ZP-424] – Merge into 2.0.x: Some folders do not have a last synchronization time
[ZP-431] – Merge into 2.0.x: Null attachments flood the log file
[ZP-432] – Merge into 2.0.x: Logging may use too much memory
[ZP-438] – Non stop sync if no changes on empty sync

Zarafa
