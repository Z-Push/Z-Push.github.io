News | Z-Push https://z-push.org Open source push technology Fri, 20 Oct 2017 12:05:42 +0000 en-US hourly 1 https://wordpress.org/?v=5.6.10 Z-Push 2.4.0 alpha1 released https://z-push.org/z-push-2-4-0-alpha1-released/ https://z-push.org/z-push-2-4-0-alpha1-released/#comments Fri, 20 Oct 2017 12:04:36 +0000 http://z-push.org/?p=579 Hello,

we have just released Z-Push 2.4.0 alpha1 (technical preview) (tag 2.4.0.alpha1 commit fc9858346a48b0b43278c2598fd0f2a4938147a7) and would like to invite you to try it.

Main feature of this version (so far) is the support of ActiveSync 14.1.
This new ActiveSync version has features which are implemented as stubs, like RightsManagement, but no backend uses this functionality already.

With the Kopano backend the GAB search shows you saved pictures of users on the mobile now for devices which support it (iOS and some Android devices).

This version now also brings configuration packages for NGINX that also supports PHP-FPM.

Some smaller performance improvements.

Complete list of the changes is available here.

In order to try this version, just switch to our develop repository and install/update your server.
More information about installing from repositories is available at the Z-Push Wiki.

Z-Push 2.4.0 will not be released as tarball anymore.
If using the repositories is not an option, it is possible to check out the code via git and the tag 2.4.0.alpha1. More information is available here.

Attention: This is an alpha version and not suitable for production usage by the general public.

Please share your experiences (positive and negative) and give feedback!

Greetings,
Sebastian
