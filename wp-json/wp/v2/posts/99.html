Hi all,<\/p>\n

we just released Z-Push 2.0 RC (build 1333).<\/p>\n

Notes<\/strong>
\nThis version fixes issues with Playbook’s and Androids’ (>=4.0.3) calendar. It also fixes the issue that a forwarded message has been truncated.
\nThe server-side search has been improved as well. The range and the folder in which the search was initiated are taken into account.
\nAndroid 4 compatibility has been improved a lot with this release.
\nAlso a lot of other smaller fixes and improvements have been made.<\/p>\n

Additional notes<\/strong>
\nThe default state directory is \/var\/lib\/z-push, the default log directory \/var\/log\/z-push.
\nPlease create these directories and make them writable for Apache
\nor change the configuration parameters to fit your needs.<\/p>\n

Be aware that if you are using LOGLEVEL_DEBUG and LOGLEVEL_WBXML Z-Push will be quite talkative,
\nso it is advisable to use log-rotate on the log file.<\/p>\n

The version is available at:\u00a0http:\/\/z-push.sourceforge.net\/download<\/a><\/p>\n

Your feedback is highly appreciated and requested.<\/p>\n

Kind regards,
\nManfred
