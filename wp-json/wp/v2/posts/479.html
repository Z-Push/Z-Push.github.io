Hi all,<\/p>\n

we have just released Z-Push 2.3.4 final version (tag 2.3.4, commit cb60b14dacebef1aefba404a1b05810e3b1d27b2).<\/p>\n

This version is PHP 7 compatible and introduces many other important changes.
\nThe PHP 7 compatibility is focussed on the main Z-Push code, there can be issues with specific backends or backend functionality.
\nWe would like to thank all our community contributors and everyone who tests Z-Push and provides us feedback. Please continue doing that.<\/p>\n

Important notes<\/strong>
\nThe PHP 7 compatibility is focussed on the main Z-Push code, there can be issues with specific backends or backend functionality. For Kopano you require Kopano Core 8.2 which is not yet released as stable.<\/p>\n

Z-Push 2.3.2 already contains fixes related to Meeting Requests on iOS. While this fix works, it’s not backward compatible to ZCP 7.1.x and 7.2.x before 7.2.3 versions.
\nThis release fixes it also for these older versions.<\/p>\n

We did introduce two new configuration options and did some changes in the headers. These will require you to update your configuration files.<\/p>\n

This version is required for KOE 1.2 which will released soon. It contains an improved API to open shared folders, also in a hierarchical expected way.
\nIt’s now also possible to stop synchronizing hidden GAB objects to the KOE GAB. See here how to do this: https:\/\/wiki.z-hub.io\/display\/ZP\/Kopano+OL+Extension#KopanoOLExtension-HiddenGABentries<\/a><\/p>\n

The full changelog is available here<\/a>.<\/p>\n
