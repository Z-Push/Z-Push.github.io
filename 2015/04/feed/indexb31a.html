today we have released the Z-Push 2.2.1 final version (build 1939).

Important notes
This version contains several bug fixes.
Appointments with “Working elsewhere” status set in Outlook 2013 should be synchronized correctly to mobile devices.
The issue that meetings have been cancelled when removing an attendee should also be fixed.

Changes since Z-Push 2.2.0
General
[ZP-570] – add an additional space after the username (and also add this space to the column header)
[ZP-575] – add working elsewhere status 4 to definitions, add conversion to busy status -1 to 1

Zarafa
[ZP-581] – Meeting request cancelled after removing attendee

The version is available at: http://download.z-push.org/final/2.2/.

Your feedback is highly appreciated.

Kind regards,
Manfred
Z-Push dev team
