we have just released Z-Push 2.3.6 final (tag 2.3.6, commit ca1e816ee03c6c3ba4ed8d439ad7478b69f5404f).
\nIt is the same as 2.3.6beta1.<\/p>\n

\nThanks guys!
\nThis version also contains a few fixes for the Kopano backend and KOE.<\/p>\n
