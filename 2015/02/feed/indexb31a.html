News | Z-Push https://z-push.org Open source push technology Wed, 11 Feb 2015 17:24:32 +0000 en-US hourly 1 https://wordpress.org/?v=5.6.10 Z-Push 2.2 final released https://z-push.org/z-push-2-2-final-released/ https://z-push.org/z-push-2-2-final-released/#comments Wed, 11 Feb 2015 17:12:04 +0000 http://z-push.org/?p=320 Hi all, today we have released the Z-Push 2.2 final version (build 1934). Important notes This version adds some new features. First is the support for autodiscover which will ease the configuration of the mobile devices. Please refer to INSTALL file in the autodiscover folder for configuration. We made some improvements for FolderSync runtime for Zarafa backend. It is optimized for a ZCP version 7.1.11 or later. The correct functionality with an earlier ZCP version is not guaranteed. We also added chunked folder sync support. It should solve the issue that the sync with android devices wasn't possible for users with a lot of (several hundreds) folders. Please read the notes in the config.php for USE_PARTIAL_FOLDERSYNC. This feature is disabled per default. Also some minor fixes like appointments with "working elsewhere" status weren't synced to the mobile device, removed svn executable property from several files. Changes since Z-Push 2.1.3 General [ZP-42] - Z-Push AutoDiscover [ZP-543] - Remove svn executable property from files Zarafa [ZP-269] - Look for enabled/disabled z-push feature in ZCP [ZP-510] - Showing appointment in Outlook as "working elsewhere" removes appointment from agenda [ZP-567] - Cache store and inbox properties for FolderSync The version is available at: http://download.z-push.org/final/2.2/.
