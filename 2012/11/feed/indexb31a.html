today we have released the final version of Z-Push 2.0.5 (build 1541). It is identical to Z-Push 2.0.5RC2.

Notes

It fixes the IOS6 meeting “hijacking” issue (read more athttp://www.zarafa.com/blog/post/2012/10/ios6-meeting-hijacking-fixed-open-source-activesync-implementation-z-push). It fixes issues with meeting requests and meetings with attendees for iOS and Outlook XP. It also fixes that “out-of-sync-window” appointments are deleted in Zarafa when using HTC devices. Droid 3 is now able to get the full message and Nokia N9MeeGo is able to download attachments.

Fileas default order for contacts created or edited on the mobile is now “Lastname, Firstname” (this setting is for displaying in outlook/webaccess only, mobile devices show contacts depending on the setting on a device).
