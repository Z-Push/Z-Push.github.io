Z-Push 2.2.13 final released

by Admin | Oct 24, 2016 | News | 0 comments

Hi all,

today we have released the Z-Push 2.2.13 version (tag 2.2.13, commit 161fe5a83152833ee62fca8f48a799458ccf567b).
Once again we thank all contributors for their work on this release.

Release notes

Meeting requests on iOS
We got reports that with the recently released iOS 10, meeting requests could not be accepted anymore.
It seems that Apple decided to introduce a breaking change (probably since iOS 9.3.x) compromising this functionality.
This version implements a workaround for these devices.

Some other improvements.

The full change log is available here.

If you are interested in a detailed overview of the code changes, see https://stash.z-hub.io/projects/ZP/repos/z-push/compare/diff?targetBranch=refs%2Ftags%2F2.2.12&sourceBranch=refs%2Ftags%2F2.2.13

The version is available at: http://download.z-push.org/final/2.2/.

Your feedback is highly appreciated.

Kind regards,
Manfred
Z-Push dev team
