Hi all,<\/p>\n

today we have released the final version of Z-Push 2.0.5 (build 1541). It is identical to Z-Push 2.0.5RC2.<\/p>\n

Notes<\/strong><\/p>\n

It fixes the IOS6 meeting “hijacking” issue (read more athttp:\/\/www.zarafa.com\/blog\/post\/2012\/10\/ios6-meeting-hijacking-fixed-open-source-activesync-implementation-z-push<\/a>). It fixes issues with meeting requests and meetings with attendees for iOS and Outlook XP. It also fixes that “out-of-sync-window” appointments are deleted in Zarafa when using HTC devices. Droid 3 is now able to get the full message and Nokia N9MeeGo is able to download attachments.<\/p>\n

Fileas default order for contacts created or edited on the mobile is now “Lastname, Firstname” (this setting is for displaying in outlook\/webaccess only, mobile devices show contacts depending on the setting on a device).<\/p>\n

Complete change log is available at:\u00a0http:\/\/zarafa-deutschland.de\/z-push-download\/final\/2.0\/z-push-2.0.5-1541.txt<\/a><\/p>\n

Your feedback is highly appreciated.<\/p>\n

Kind regards,<\/p>\n
