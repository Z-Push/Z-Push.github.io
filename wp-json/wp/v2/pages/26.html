    1. It isn’t working!<\/strong><\/p>\n

    Please check at first, if your webserver is working correctly. Open the following URL in your web browser:<\/p>\n

    https:\/\/[your-server-name]\/Microsoft-Server-ActiveSync<\/pre>\n
    An authentication dialog should be shown. You should be able to log in with your normal credentials.<\/p>\n
    If the dialog isn’t shown, something is wrong with your installation. Re-check the instructions in the INSTALL file or ask in the\u00a0forum<\/a>.<\/p>\n
    2. Does it work with my device?<\/strong><\/p>\n
    In general, Z-Push works with all ActiveSync compatible mobile devices, including Windows Phones, iPhones, Android, Sony Ericsson and Nokia mobile devices. A working internet connection to your Z-Push enabled server is necessary.<\/p>\n
    Detailed information about devices and compatibility status can be found in the Mobile Compatibility List<\/a>.<\/p>\n
    You are welcome to give feedback and suggestions about the device compatibility so we can improve the compatibility list.<\/p>\n
    3. How do I enable the WBXML log?<\/strong><\/p>\n
    In Z-Push 2.X versions set the value of LOGLEVEL parameter in the config to LOGLEVEL_WBXML. The log file stays the same.<\/p>\n","protected":false},"excerpt":{"rendered":"
