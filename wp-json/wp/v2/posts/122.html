Hi all,<\/p>\n

today we have released the Z-Push 2.0.8 final version (build 1731).<\/p>\n

Notes<\/strong>
\nThis version contains several fixes. Changing note of an occurrence of a
\nrecurring appointment now is synced to the mobile. Also an issue that the
\ntracking tab in the WebApp was missing for meetings created on iOS devices
\nhas been resolved.
\nEmails flagged on an Iphone are now correctly displayed in the task view.
\nReminder is no longer active for completed tasks.
\nIOS devices now are able to download signed emails with large attachments.
\nBetter loging handling.<\/p>\n

Changes<\/strong><\/p>\n

General
\n[ZP-419] – Typo in syncappointment.php
\n[ZP-424] – Merge into 2.0.x: Some folders do not have a last synchronization time
\n[ZP-431] – Merge into 2.0.x: Null attachments flood the log file
\n[ZP-432] – Merge into 2.0.x: Logging may use too much memory
\n[ZP-438] – Non stop sync if no changes on empty sync<\/p>\n

Zarafa
\n[ZP-418] – Changed note of an occurrence of a recurring appointment won’t be synced to the mobile
\n[ZP-436] – Issues with appointments on IOS 6.1.3
\n[ZP-441] – Email flagged on an iPhone is without subject in task view
\n[ZP-442] – Reminder still active for completed tasks
\n[ZP-445] – S\/MIME Mails with larger attachments are not able to be fully downloaded (on IOS)<\/p>\n

Your feedback is highly appreciated.<\/p>\n

The download is available at:\u00a0http:\/\/zarafa-deutschland.de\/z-push-download\/final\/2.0\/<\/a><\/p>\n
