News | Z-Push https://z-push.org Open source push technology Sun, 02 Feb 2014 12:51:27 +0000 en-US hourly 1 https://wordpress.org/?v=5.6.10 Z-Push 2.0.5RC released https://z-push.org/z-push-2-0-5rc-released/ https://z-push.org/z-push-2-0-5rc-released/#respond Thu, 18 Oct 2012 12:00:20 +0000 http://62.245.227.250/?p=113 Hi all,

today we have released the Z-Push 2.0.5RC version (build 1511).

Notes
It fixes the IOS6 meeting “hijacking” issue (read more athttp://www.zarafa.com/blog/post/2012/10/ios6-meeting-hijacking-fixed-open-source-activesync-implementation-z-push). It also fixes that “out-of-sync-window” appointments are deleted in Zarafa when using HTC devices. Droid 3 is now able to get the full message and Nokia N9MeeGo is able to download attachments.

Fileas default order for contacts created or edited on the mobile is now “Lastname, Firstname” (this setting is for displaying in outlook/webaccess only, mobile devices show contacts depending on the setting on a device).

The version is available at: http://z-push.sourceforge.net/download.

Complete change log is available at: http://zarafa-deutschland.de/z-push-download/beta/2.0/z-push-2.0.5RC-1511.txt

Your feedback is highly appreciated.

Kind regards,
Manfred
Z-Push dev team
