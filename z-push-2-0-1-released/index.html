Z-Push 2.0.1 released

by Manfred | Jul 12, 2012 | News | 0 comments

Hi all,

today we have released the final version of Z-Push 2.0.1 (build 1375).

Notes

This version fixes the issue that the description of an appointment on Windows Mobile was not fully downloaded.
It also fixes that Móxier Mail for Android was not able to sync folder contents.
Also several smaller issues have been fixed.

The complete changelog is available at: http://zarafa-deutschland.de/z-push-download/final/2.0/z-push-2.0.1-1375.txt.

The version is available at: http://z-push.sourceforge.net/download.

Your feedback is highly appreciated and requested.

Kind regards,
Manfred
Z-Push dev team
