Hi all,<\/p>\n

we have just released Z-Push 2.3.2 final (tag 2.3.2, commit f9ac780ee5aadde4b17d04429e90d6d33405fd1a).<\/p>\n

This version contains two major changes:
\nMeeting requests on iOS<\/strong>
\nWe got reports that with the recently released iOS 10, meeting requests could not be accepted anymore.
\nIt seems that Apple decided to introduce a breaking change (probably since iOS 9.3.x) compromising this functionality.
\nThis version implements a workaround for these devices.<\/p>\n

Send-as functionality<\/strong>
\nEspecially the Kopano Outlook Extension users requested a way also to use “Send-As” from Outlook.
\nThe required changes for Z-Push are already in this version, but the full feature is still under development for KOE.
\nIt will also be possible to use send-as on a mobile phone, but only if you reply to an email received in a shared folder.
\nWhen opening the shared folder on your phone, you can define that all replies should be send-as the owner of the store.
\nThis is fully functional, but currently there is no convinient way to set this up.
\nThis functionality is still disabled by default. Check ticket ZP-908 on instructions how to enable\/use this feature.<\/p>\n

We would like to thank all our community contributors and everyone who tests Z-Push and provides us feedback. Please continue doing that.<\/p>\n
