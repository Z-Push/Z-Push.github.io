News | Z-Push https://z-push.org Open source push technology Wed, 27 Jan 2016 17:38:49 +0000 en-US hourly 1 https://wordpress.org/?v=5.6.10 Z-Push 2.2.8 final released https://z-push.org/z-push-2-2-8-final-released/ https://z-push.org/z-push-2-2-8-final-released/#comments Wed, 27 Jan 2016 17:38:49 +0000 http://z-push.org/?p=363 Hi all,

today we have released the Z-Push 2.2.8 final version (tag 2.2.8, commit 208e6bbb8e38f4211b4a0afa077a3f8c4dd16163). Once again we thank all contributors for their work on this release.

Important notes
This version contains some fixes and improvements.
The problem with the device trying to remove a message which was already removed on the server has been resolved.
This version also fixes the issue with LoopDetection getting stuck in some cases.
We added a backwards compatibility for ZCP 7.0.X versions and improved conflict detection for the Zarafa backend.
Autodiscover won’t generate a lot of warnings for PHP 5.4 and higher because of the timezone.

The full change log is available here.

The version is available at: http://download.z-push.org/final/2.2/.

Your feedback is highly appreciated.

Kind regards,
Manfred
Z-Push dev team
